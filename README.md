# Programa-o-Competitiva
Aqui irei colocar minhas resoluções de questões de programação competitiva como forma de aprendizagem.

## Material de estudo:
Estou utilizando o material de estudo do [Monkeys](https://monkeys-ufg.github.io/website/) -> Grupo de Programção Competitiva da Universidade Federal de Goiás.

## Porque iniciei?
Eu acho muito divertido resolver problemas e poder aplicar o pensamento crítico, de modo que utilize as ferramentas que aprendi sobre da forma mais eficiente.
Para que possa, quem sabe, participar de competições de verdade, como por exemplo a OBI.
